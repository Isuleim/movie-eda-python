# movie-eda-python
This data science project explores movie trends from the 1990s using Python and pandas. It applies exploratory data analysis techniques such as filtering, grouping, visualization, and statistical analysis to identify runtime patterns and insights from real-world entertainment data.
🎬 Movie Runtime Analysis (1990s)
📌 Overview

This project explores movie trends from the 1990s using Python and pandas. It applies exploratory data analysis (EDA) techniques to understand patterns in movie durations and genres.

📊 Key Insights
Most common movie duration in the 1990s: 94 minutes
Only 7 action movies were under 90 minutes
Movie runtimes in this era show a consistent industry standard
